# Directorio de Fotos de Autores

Este directorio contiene las fotos de perfil de los autores del blog.

## Cómo usar

1. Sube la foto del autor a este directorio
2. Usa un nombre descriptivo, por ejemplo: `jesus-florez.jpg`
3. En el markdown del post, referencia la foto: `fotoAutor: "/authors/jesus-florez.jpg"`

## Formatos soportados

- JPG/JPEG
- PNG
- WebP (recomendado para mejor rendimiento)

## Recomendaciones

- **Tamaño recomendado**: 200x200px o 300x300px
- **Peso máximo**: 100KB
- **Formato cuadrado** para mejor visualización en avatar circular
- Usa herramientas como TinyPNG o Squoosh para comprimir las imágenes

## Ejemplo de uso en Markdown

```markdown
---
titulo: "Mi Post"
fecha: "2026-01-12"
nombreAutor: "Jesús Flórez"
fotoAutor: "/authors/jesus-florez.jpg"
---
```
